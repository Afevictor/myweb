# Images Folder

This folder contains images for the Verturn Technologies website.

## Add Your Photo

To display your photo in the "Meet the Founder" section:

1. Add your photo to this folder
2. Name it: `victor-afe.jpg`
3. Recommended size: 400x400 pixels or larger (square format works best)
4. Supported formats: .jpg, .jpeg, .png, .webp

The image will automatically appear on the homepage in the "Meet the Founder" section.
